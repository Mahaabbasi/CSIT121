About Me

Welcome to my portfolio! I'm Maha Abbasi, a student pursuing web development. This semester has been an incredible learning journey where I’ve developed new skills in HTML, CSS, and responsive design.

I have created several projects that helped me grow as a developer. Each task presented new challenges, but the satisfaction of seeing the final result always made it worthwhile. Through these projects, I’ve gained a solid understanding of web development concepts and techniques.

Lessons Learned

Completing this project was both challenging and rewarding. One of my main challenges was working with CSS to achieve the right layout. I struggled with positioning elements correctly and ensuring a responsive design, but with practice, I gained more confidence.

Seeing my website come together was a major triumph. The process of experimenting with different layouts and refining the code taught me how to approach web design with flexibility and patience.

Testing on multiple browsers and devices was crucial to ensure my website worked across all platforms. Initially, I faced layout issues when testing on Firefox and Safari, which taught me the importance of browser compatibility.

I also realized the value of mobile-first design. Ensuring that the website was responsive and functional on both mobile and desktop devices became a key focus as I worked through the project.

This experience reinforced that testing and debugging are essential parts of the web development process. It also taught me that a website should be consistent and functional across different screen sizes and browsers.

Throughout this course, I have learned the importance of clean, semantic code and the significance of designing for accessibility. Understanding how to make websites responsive and user-friendly has been one of the biggest takeaways.

Above all, this course has given me a deeper appreciation for the craft of web development. I’ve not only learned technical skills but also the importance of user experience and design thinking in creating engaging, functional websites.

Key Skills Learned

  • Responsive Web Design: Ensuring websites are adaptable to different screen sizes and devices.
  • CSS Flexbox and Grid: Mastering layout techniques to create dynamic and flexible designs.
  • Browser Compatibility: Testing and ensuring that websites function well across multiple browsers.
