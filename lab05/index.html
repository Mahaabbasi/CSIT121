Labs 1 & 2 Reflection

Navigating the Challenges of Building My First Webpage

Creating my first webpage was quite challenging at the start due to the many HTML tags and layout rules to learn. It felt overwhelming with so much information to take in all at once. However, by breaking the task into smaller, more manageable steps, I found it easier to understand. Gradually, I learned how to use HTML tags correctly and how to structure a webpage. This methodical approach helped me see how all the parts fit together and how webpages are built and displayed on the internet.

Through persistence and practice, I gained confidence in my web development skills.

Lab 03 Reflection

In Lab 3, I deepened my understanding of responsive design and CSS. Here are three key takeaways:

  1. I learned why semantic HTML is important and how using meaningful HTML tags makes websites more accessible to everyone and helps them rank better in search engines.
  2. I learned how to use HTML elements effectively by practicing with headers, sections, and articles to create a clear and well-organized webpage.
  3. I learned how to enhance webpages with CSS by discovering how to improve their appearance while ensuring they remain easy to use and have a consistent design.

Lab 05 Reflection

In Lab 5, I enhanced my skills in creating multi-column layouts using CSS Grid. Here are three key takeaways:

  1. I improved my skills in creating multi-column layouts using CSS Grid.
  2. Implementing accessibility practices in navigation and structure enhanced the overall user experience.
  3. Using consistent styling and spacing made my designs cleaner and more professional.

Reflection on Best Design Practices

In creating responsive designs, it's crucial to prioritize usability and accessibility. This includes ensuring that text is legible on all devices, navigation is intuitive, and layouts adapt fluidly to various screen sizes. Using semantic HTML and structured CSS is essential in creating clean, maintainable code that enhances both user experience and search engine optimization.

Additionally, consistency in design elements such as colors, fonts, and spacing improves familiarity and comfort for users. Testing designs on various devices and browsers ensures that all users have a positive experience.
